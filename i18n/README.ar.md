## لماذا ClaudeKit؟

Claude Code أداة قوية بمفردها. ClaudeKit يجعلها **منظمة وآمنة وقابلة للتدقيق**.

بدون ClaudeKit، يُجري المساعد الذكي تغييرات مباشرة — بدون خطة، بدون مراجعة، بدون تراجع. مع ClaudeKit، كل تغيير يتبع خط أنابيب: خطط له، راجعه، نفذه بأمان، تحقق من النتيجة.

## البداية السريعة

### التثبيت

```bash
git clone https://github.com/OmarMokhtar-Saad/claudekit.git
./claudekit/install.sh /path/to/your-project --full
```

يقوم المُثبّت بالكشف التلقائي عن لغة مشروعك، ونسخ مجلد `.claude/` إلى مشروعك، وإنشاء ملفات `CLAUDE.md` و `CONSTITUTION.md`، وتكوين الخطافات مع أوامر البناء والاختبار والفحص.

### خيارات التثبيت

```bash
# تثبيت كامل (وكلاء + أوامر + مهارات + خطافات + عمليات)
./install.sh ./my-project --full

# تثبيت مختصر (وكلاء + أوامر + عمليات فقط)
./install.sh ./my-project --minimal

# تحديد اللغة مسبقاً
./install.sh ./my-project --full --language typescript

# الكتابة فوق تثبيت موجود
./install.sh ./my-project --full --force
```

### الاستخدام

افتح مشروعك في Claude Code ونفّذ:

```
/plan إضافة مصادقة المستخدم باستخدام JWT
```

يتولى ClaudeKit العملية — يستكشف المخطط قاعدة الكود، يكتب خطة مع تكوين ops.json، يتحقق المراجع منها، ينفذها المنفذ مع نسخ احتياطية تلقائية، ويفحص المدقق النتيجة.

## سير العمل المبني على المواصفات

1. اكتب مواصفة في `specs/`
2. نفّذ `/plan` للتخطيط من المواصفة
3. يقوم المراجع بالتحقق مقابل المواصفة
4. يضمن المدقق التوافق مع المواصفة
